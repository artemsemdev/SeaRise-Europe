# Code of Conduct

## Our Standard

Participation in this repository should be respectful, direct, and evidence-based.

Expected behavior:

- Be respectful in disagreement.
- Critique ideas, decisions, and implementations, not people.
- Prefer clear technical reasoning over sarcasm or personal attacks.
- Help keep the repository useful for contributors, reviewers, and learners.

Unacceptable behavior:

- Harassment, hate speech, or discriminatory language
- Personal attacks or intimidation
- Doxxing or sharing private information without consent
- Sexualized language or unwanted sexual attention
- Repeated bad-faith disruption of discussion or review

## Enforcement

The maintainer may remove comments, close discussions, reject contributions, or block participation when behavior violates this code of conduct.

Possible responses include:

- a private warning
- temporary restriction from discussions or pull requests
- permanent removal from the project space

## Reporting

If you experience or observe unacceptable behavior, report it privately to the maintainer through GitHub rather than escalating it in public threads.

## Scope

This code of conduct applies to issues, pull requests, discussions, code review, and other project interactions in this repository.
