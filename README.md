# SeaRise Europe

SeaRise Europe is a public, documentation-first repository for a Europe-only web application that aims to make scenario-based coastal sea-level exposure understandable without overstating what the science can say.

The repository is currently in the planning and architecture phase. There is no runnable application in this repo yet. The current deliverables are the product definition and the technical architecture that will guide implementation.

## Repository Status

- Phase: product definition and architecture
- Current contents: product docs and proposed architecture
- Planned implementation: Next.js frontend, ASP.NET Core API, TiTiler, PostgreSQL/PostGIS, Azure Blob Storage, and a Python/GDAL geospatial pipeline

## What This Project Is

- A portfolio-grade, public planning repository
- A docs-first foundation for a future implementation
- A focused product: Europe, coastal exposure, scenario-based time horizons

## What This Project Is Not

- A live public service
- An engineering, insurance, mortgage, or legal risk tool
- A parcel-level guarantee or real-time flood forecasting product
