Happy Path
Landing → Search Loading → Candidates → Assessing → Exposure Detected
01 Landing Hero search with Europe map background. Entry point for all flows. 02 Search Loading Spinner state while geocoding the user's search query. 03 Candidates Dropdown list of matching locations for the user to pick from. 05 Assessing Map zooms in while sea-level exposure is being calculated. 06 Exposure Detected Full result with sidebar controls, exposure zone overlay, and data card.
Result States
06 Exposure Detected Main result showing sea-level rise risk with projected flood zone. 07 No Exposure Safe location with no projected sea-level exposure. 08 Data Unavailable Coastal location but no projection data available yet. 09 Inland / Out of Scope Location is too far from the coast for sea-level analysis. 10 Outside Europe Location is outside the European coverage area.
